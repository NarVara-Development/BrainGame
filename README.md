# 🧠 NarVara BrainGame

Aplikasi latihan otak **10-in-1** dengan tampilan brutalist yang bold & penuh animasi. Lebih dari **50.000 soal** prosedural — main berbulan-bulan tanpa bosan.

## 🎮 Games
Logika · Matematika · Kata · Teka-Teki Silang · Memori · Number Tap · Benar/Salah · Slide Puzzle · Odd One Out · Simon

## ✨ Fitur
- 50.000+ soal, tingkat kesulitan adaptif
- Leaderboard global, statistik & streak harian
- Login Email / Google / Tamu
- Bahasa Indonesia & English, bisa offline
- Premium (sekali bayar) untuk hapus iklan

## 🛠️ Tech Stack
- **React Native** (Expo SDK 56, Expo Router)
- **Zustand** (state), **Supabase** (auth + DB)
- **Google AdMob** (banner + rewarded video), **react-native-iap** (Play Billing)
- **Firebase Analytics**, **Groq AI** (soal tambahan)

## 🚀 Menjalankan

```bash
npm install
npx expo start
```

> Butuh file `.env` (Supabase, Groq, AdMob) — tidak disertakan demi keamanan.
